Bitácora

Semana 1

En esta semana definimos a los Equipos dentro del salón de clases y compartimos opiniones sobre la problemática que íbamos a escoger. Al final decidimos guiarnos por la inseguridad en Lima y ya teniendo definido eso comenzamos a realizar las encuestas. Con esto llegamos a la conclusión que la encuesta revela que la mayoría de los encuestados desean una alarma antirrobo invisible y altamente confiable. También se realizo un Lean Canvas de nuestra solución que sacamos de nuestra investigación previa.

Semana 2

En la semana 2 el profesor nos enseñó a usar lo básico en el Visual Studio Code para tener un conocimiento previo de como realizar nuestra propia página web en la que estaría nuestra bitácora digital. También nos enseñó como usar el Github, este nos ayudaría a subir nuestra página a la web y también guardar los cambios que realicemos vinculándola con el Visual Studio Code.

Semana 3

En esta semana, al concluir nuestra investigación, comenzamos a idear posibles soluciones para abordar nuestra problemática. Para este fin, aplicamos diversas herramientas, entre las cuales se incluyeron varias tablas que nos fueron enseñadas durante el proceso. Utilizamos estas tablas para organizar nuestras ideas y evaluar su viabilidad y nivel de innovación, luego se escogió la mejor idea para poder conceptualizarla.
